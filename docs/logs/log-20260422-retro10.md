- 이번 장애는 zip 생성 측 Maven BOM 해석 실패로, metadata 조회는 정상이었을 가능성이 높다.
- SKILL.md §2 는 두 실패를 구분하지 않고 "접근 실패 시 중단" 으로만 기술 → 테스터에게
  선택지가 "전체 중단" 혹은 "하드코딩 우회" 양극단만 남았다.
- 절대 규칙 #1 재진술이 §2 본문에 없어 압박 상황에서 우회 심리적 장벽이 낮았다.
